# AERIA Prompt Library  
## Declaración sobre cómo usar IA con criterio

La mayoría de los prompts buscan resultados.  
Este repositorio busca dirección.

Hoy es posible generar cientos de imágenes, textos o variaciones en minutos.  
Pero generar no es diseñar.

Un prompt sin marco es solo instrucción técnica.  
Un prompt con intención es una decisión estructural.

---

## Qué es esta librería

Una colección curada de estructuras de prompt diseñadas desde:

- Tensión conceptual  
- Restricción estratégica  
- Coherencia visual  
- Pensamiento crítico  

No son recetas cerradas.  
Son marcos para pensar.

Cada estructura aquí compartida incluye:

- Contexto
- Tensión
- Restricciones
- Prompt base
- Criterios de evaluación

Porque el resultado no depende del modelo.  
Depende del sistema que lo sostiene.

---

## Qué no es

- No es un repositorio de “100 prompts virales”.
- No es una colección de hacks.
- No es un atajo para producir más rápido.

Si buscás efectos inmediatos, este no es el lugar.

---

## Principio central

La IA amplifica.  
El criterio dirige.

Sin marco conceptual, la automatización multiplica ruido.  
Con marco conceptual, la automatización amplifica identidad.

---

## Cómo usar esta librería

1. No copies sin entender la tensión.
2. No elimines las restricciones.
3. No evalúes solo por impacto visual.
4. Iterá con criterio, no por impulso.

Cada estructura está pensada como punto de partida.  
Adaptá. Ajustá. Documentá.

---

## Por qué existe

Este repositorio forma parte de una práctica más amplia:

Diseñar coherencia estructural en sistemas creativos que integran identidad, cultura y tecnología.

Menos ruido.  
Más dirección.
